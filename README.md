coapp-vsp-prototype
===================

Visual Studio plugin for CoApp packages

![Package Manager](coapp-vsp-prototype/raw/master/pkgmgr.png)

### Notes

- Requires CoApp.Toolkit 1.2.0.165 (for now).

- Set startup project to "Loader" if it isn't.
